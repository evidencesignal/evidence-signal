# evidence-signal

Python package scaffold: extract and reason about signals from evidence. Requires Python 3.9+.

## Setup

```bash
cd evidence-signal
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
```

## Run tests

```bash
pytest
```

## Layout

- `src/evidence_signal/` — package code
- `tests/` — pytest tests
